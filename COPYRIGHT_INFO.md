### How It Works

This application uses **procedural audio synthesis** to create sounds in real-time. No pre-recorded samples are used by default. Every sound is generated mathematically using:

- **Oscillators** - Generate waveforms (sine, sawtooth, square, triangle)
- **Filters** - Shape the frequency content
- **Envelopes** - Control volume over time
- **Noise Generators** - Create percussive textures
